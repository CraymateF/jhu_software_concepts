================================================================================
CUSTOMIZATION GUIDE:
================================================================================

To personalize your portfolio:

1. EDIT HOMEPAGE CONTENT:
   File: app/templates/home.html
   - Change "Your Name" to your actual name
   - Update "Your Position / Title"
   - Replace bio text
   - Add your profile picture to app/static/images/profile.jpg

2. EDIT CONTACT INFO:
   File: app/templates/contact.html
   - Update email address
   - Update LinkedIn URL
   - Update GitHub URL
   - Add phone number if desired

3. UPDATE PROJECTS:
   File: app/templates/projects.html
   - Update project title
   - Modify project description
   - Update GitHub repository link

4. CHANGE COLORS:
   File: app/static/css/style.css
   - Edit :root CSS variables at top of file:
     * --primary-color: Main text color
     * --secondary-color: Highlight/button color
     * --nav-bg: Navigation bar background

5. ADD MORE PAGES:
   - Create new blueprint in app/routes/
   - Create new template in app/templates/
   - Register in app/__init__.py
   - Add link in base.html navigation

================================================================================
DEPLOYMENT NOTES:
================================================================================

For production deployment, follow these steps:

1. Set debug=False in run.py
2. Use Gunicorn instead of Flask development server:
   $ pip install gunicorn
   $ gunicorn -w 4 -b 0.0.0.0:8080 'app:create_app()'
3. Use a reverse proxy (Nginx or Apache)
4. Enable HTTPS/SSL
5. Use environment variables for secrets
